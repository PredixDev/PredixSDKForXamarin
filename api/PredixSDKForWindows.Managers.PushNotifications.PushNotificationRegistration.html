Class PushNotificationRegistration

Handles registering device for push notifications on the Predix Push Notification service

Example usage:

var pushRegistration = new PushNotificationRegistation(token: deviceToken, deviceId: deviceId)
await pushRegistration.RegisterAsync(pushServiceURL, myApplicationKey)

PushNotificationRegistration(String, String, String)

Initializes a new PushNotificationRegistation structure, for registering this device and user with Push Notifications

This service should be initizalized in the platform specific projects as token, deviceId, and appKey are properties specific to each platform.

Push Notification registration token from the device.
