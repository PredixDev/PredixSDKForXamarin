OpenDatabaseConfiguration(String, Uri, IEnumerable<IIndexer>)

Initializes a DatabaseConfiguration

  • important: Database names are restricted to using only the following characters: abcdefghijklmnopqrstuvwxyz0123456789_$()+-/

  • parameter name: Name of database. If nil, the value returned from defaultDatabaseName is used. Database names are restricted to using only the following characters: abcdefghijklmnopqrstuvwxyz0123456789_$()+-/

  • parameter fileLocation: Local file URL of database files. If nil, the value returned from defaultLocation is used.

  • parameter completionQueue: An operation queue for scheduling completion handlers. Defaults to the main queue
