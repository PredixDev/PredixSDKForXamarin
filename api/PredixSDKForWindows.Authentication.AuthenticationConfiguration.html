AllowAuthenticationTokenForPredixUrlSessions

Allows the authentication manager to set a UAA Authentication token as the global token to use for Predix session handlers.

Declaration
public bool AllowAuthenticationTokenForPredixUrlSessions { get; set; }
Property Value
Type Description
System.Boolean

true if allow authentication token for predix URL sessions; otherwise, false.
Remarks

Setting a token as the authentication token for PredixUrlSessions allows url sessions and PredixRequestMessage created to automaticlly populate the Authentication header with the token recieved during a successful authentication.

AllowOnlineFailureFallbackToOffline

If true, then in the event of a failure during online authentication, the system will attempt offline authentication.

Declaration
public bool AllowOnlineFailureFallbackToOffline { get; set; }
Property Value
Type Description
System.Boolean

true if allowing fallback to offline login when online authentication fails, else false
Remarks

The fallback to offline only occurs if the online authentication responds with a Fail. Canceled or Denied do not fallback.
